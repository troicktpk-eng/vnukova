# Movie Library — Личная кинотека

**Автор:** [Ваше Имя Фамилия]
**Язык:** Python 3.x

## Описание

Графическое приложение для хранения и фильтрации личной коллекции фильмов. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Данные сохраняются в формате JSON. Реализована фильтрация по жанру и году.

## Технологии

- Python 3.x.
- Библиотека `tkinter` для GUI.
- JSON для хранения данных.
- Git для контроля версий.

## Запуск

1. Установите Python (>=3.6).
2. Установите зависимости: `pip install -r requirements.txt` (если требуется).
3. Запустите: `python main.py`.

## Примеры использования

1. Введите данные о фильме и нажмите «Добавить фильм».
2. Используйте фильтры для поиска по жанру или году.
3. Данные сохраняются автоматически.
